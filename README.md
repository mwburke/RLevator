# RLevator

RLevator (pronounced R elevator) is a project where I explore the use of reinforcement learning to create an automated elevator agent using python with an associated website documenting the process built using [Idyll](https://idyll-lang.org/) and [p5js](https://p5js.org/).

You can visit the writeup code in the `docs` directory and a live version at [mwburke.github.io/RLevator](http://mwburke.github.io/). The actual RL code can be found in the `rlevator` directory.
